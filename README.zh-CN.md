# ROG G614FR Ubuntu Maintenance Skill 中文说明

英文主说明请优先看：

这个仓库放的是一个公开的 Codex skill，目标机器是 ASUS ROG G614FR，
也就是国内常见的 ROG 魔霸9 这一类机型。

它主要覆盖这几类问题：

- 开机前期出现三十多秒空白或卡住
- 插着键盘开机会异常
- BIOS / GRUB 启动优化
- Ubuntu 内置喇叭无声，但有线耳机正常
- 登录后默认静音这类便利设置

- [rog-mob9-ubuntu-maintenance](./rog-mob9-ubuntu-maintenance)
